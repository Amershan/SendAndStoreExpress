# SendAndStoreExpress
Store delivery items, order them, delete them with express framework

#Install
```sh
$ git clone [git-repo-url]
$ cd SendAndStoreExpress
$ npm install 
$ if you using linux issue sudo npm install
$ npm start
$ open http://localhost:3000 in a browser 
```
